# Dados do Desafio 2

Esta versão do repositório não inclui as imagens e labels originais do Desafio 2 para manter o projeto enxuto no GitHub.

Para reexecutar o treinamento completo, recoloque os arquivos originais nestas pastas:

- `Desafio_2/data/images/`
- `Desafio_2/data/labels/`

Os labels devem seguir o formato YOLO de segmentação, com classe `0` para `fissura`.
